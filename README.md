# Dashboard_PowerBi_EcommerceBrazillian
**Link dataset : https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce**
<img width="812" height="408" alt="image" src="https://github.com/user-attachments/assets/23852742-525a-4de0-98ed-7a8380a747c0" />


Project ini merupakan pembuatan dashboard Business Intelligence berbasis Power BI untuk menganalisis performa operasional e-commerce menggunakan data transaksi nyata dari platform Olist, marketplace terbesar di Brazil. Tujuannya adalah mengubah data transaksi mentah yang tersebar di banyak tabel menjadi satu tampilan visual interaktif yang membantu pengambilan keputusan bisnis dari tiga sisi utama: performa penjualan, efisiensi logistik/pengiriman, dan kepuasan pelanggan. Alur kerja project terbagi menjadi dua tahap, yaitu pra-pemrosesan data (data cleaning) menggunakan Python di Google Colab, dan visualisasi data menggunakan Power BI untuk membangun dashboard interaktif yang dilengkapi filter dan berbagai jenis chart.

Dashboard ini dirancang untuk menjawab pertanyaan analitis seputar tren penjualan dari waktu ke waktu, efisiensi pengiriman logistik (proporsi tepat waktu, terlambat, dan tidak terkirim), kategori produk yang paling menguntungkan, preferensi metode pembayaran pelanggan, serta sebaran geografis pesanan dan kepuasan pelanggan antar wilayah. Untuk menjawab pertanyaan tersebut, dashboard menampilkan beberapa KPI utama seperti Total Revenue, Total Orders, Average Order Value, Total Freight, Average Rating, dan rata-rata durasi pengiriman (SLA). Ketiga KPI inti, yaitu Revenue, Rating, dan SLA, mewakili tiga dimensi kesehatan bisnis sekaligus: pertumbuhan finansial, kepuasan pelanggan, dan efisiensi operasional.

Secara desain, dashboard terdiri dari 23 objek visual berupa card ringkasan, line chart tren penjualan, bar/column chart status pengiriman dan kategori produk, donut chart metode pembayaran, peta sebaran geografis, gauge performa SLA, serta tiga slicer interaktif untuk kategori produk, wilayah, dan status pengiriman. Tata letaknya mengikuti alur logis dari atas ke bawah dan kiri ke kanan, dengan tema warna gelap beraksen oranye agar angka mudah dibaca. Ketiga slicer saling terhubung ke seluruh visual melalui relasi antar tabel, sehingga setiap perubahan filter langsung memengaruhi semua KPI dan chart secara real-time.

Dari hasil eksplorasi dashboard, ditemukan beberapa insight utama. Pertama, mayoritas pengiriman tercatat tepat waktu, namun rata-rata durasi SLA masih mendekati ambang batas atas sehingga proses logistik secara umum tergolong lambat. Kedua, terdapat ketergantungan tinggi pada metode pembayaran credit card dibanding metode lain seperti boleto dan voucher, yang menandakan risiko konsentrasi pada satu kanal pembayaran. Ketiga, kategori health_beauty dan watches_gifts menjadi kontributor revenue terbesar, mengindikasikan preferensi pasar terhadap produk personal-care dan lifestyle. Terakhir, tren penjualan bersifat fluktuatif dengan lonjakan pada bulan-bulan tertentu, mengindikasikan adanya pola musiman atau event promosi yang perlu diselaraskan dengan strategi stok dan logistik.
